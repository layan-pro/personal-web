Python NumberBook project

This program allows users to enter phone numbers and checks if each one meets specific conditions (10 digits and starts with “05”). If not, the program prompts the user to re-enter it. Once a valid number is entered, it requests the associated username and saves the information. When the user press “Enter” twice The program displays all saved numbers and then stops executing.
